COME USARE:

Tenere aggiornato file di testo, chiamato "machines_list.txt",
contenente la lista degli identificativi delle macchine,
situato nella cartella "res".
Per aggiungere o togliere una macchina dalla supervisione basta
aggiungere o togliere il suo identificativo dal file sopramenzionato.
N.B.: la rimozione, dei dati delle macchine rimosse, dal database vanno fatte manualmente, onde evitare perdite di dati.

Tenere aggiornato file di testo, chiamato "errors_list.txt",
contenente la lista degli identificativi degli errori,
situato nella cartella "res".
Per aggiungere o togliere un errore tra quelli possibili basta
aggiungere o togliere il suo identificativo dal file sopramenzionato.

Per riconnettere le macchine disconnesse cliccare sul bottone "reconnect".
